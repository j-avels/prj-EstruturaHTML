TAGS DE TEXTO Aqui vamos aprender sobre tags para formatação de texto. Posso escrever um texto normal. Ele não terá nenhum formato. Este é um parágrafro simples! Para pular linha você pode usar a tag . Viu só, pulou linha! Deixar o texto em negrito Em itálico Ou até sublinhado Texto tachado Texto destacado Texto menor Podemos deixar também o Texto sobrescrito Ou o Texto subescrito Ah! Temos as TAGS SEMANTICAS, que possuem algumas razões de existirem: - Ajudam o browser a entender o significado do texto - Ajudam os motores de busca a indexar o conteúdo. - Ajudam os leitores de tela a ler o conteúdo para deficientes visuais. - Ajudam os tradutores automáticos a traduzir o conteúdo. Exemplos: - Strong deixa o texto forte - Emphasis dá ênfase ao texto - Delete te diz para desconsiderar um trecho deletado - Insert diz para considerar um trecho inserido
