Tags Básicas de HTML

As tags básicas de HTML são essenciais para estruturar qualquer página web. Veja as principais:

  • <!DOCTYPE html>: Define o tipo do documento e a versão do HTML.
  • <html>: Elemento raiz que envolve todo o conteúdo da página.
  • <head>: Contém metadados, links para estilos, scripts e o título da página.
  • <meta>: Define metadados como codificação de caracteres e descrição.
  • <title>: Define o título exibido na aba do navegador.
  • <body>: Contém todo o conteúdo visível da página, como textos, imagens e links.
  • <h1>...<h6>: Títulos e subtítulos, do mais importante (h1) ao menos importante (h6).
  • <p>: Paragraph. Define um parágrafo de texto.
  • <a>: Anchor. Cria um link para outra página ou recurso.
  • <img>: Image. Insere uma imagem na página.
  • <figcaption>: Define a legenda de uma imagem.
  • <ul>, <ol> e <li>: Unorder List, Order List, List Item. Criam listas não ordenadas, ordenadas e itens de lista.
  • <div>: Division. Define uma divisão ou seção genérica no documento.
  • <span>: Marca uma extensão ou parte genérica no texto para aplicar estilos ou scripts.

Exemplo de Estrutura Básica

            
                <!DOCTYPE html>
                <html lang="pt-BR">
                <head>
                    <meta charset="UTF-8">
                    <title>Minha Página</title>
                </head>
                <body>
                    <h1>Olá, mundo!</h1>
                    <p>Este é um exemplo de página HTML.</p>
                </body>
                </html>

Tags de Tabela em HTML

As principais tags para criar tabelas em HTML são:

  • <table>: Define o início e o fim da tabela.
  • <caption>: Adiciona um título à tabela.
  • <thead>: Agrupa o cabeçalho da tabela.
  • <tbody>: Agrupa o corpo da tabela.
  • <th>: Define um cabeçalho de coluna.
  • <tr>: Define uma linha da tabela.
  • <td>: Define uma célula de dados.
  • <tfoot>: Agrupa o rodapé da tabela.
