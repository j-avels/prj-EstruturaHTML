Tags de Formulário em HTML

As tags de formulário em HTML permitem criar interfaces para entrada de dados pelo usuário. Elas são essenciais para coletar e enviar informações em páginas web, como cadastros, pesquisas e autenticação.

  • <form>: Define o início e o fim de um formulário.
    • action: URL para onde os dados do formulário serão enviados.
    • method: Método de envio dos dados (get ou post)
  • <fieldset>: É um container que agrupa elementos relacionados em um formulário.
  • <legend>: Título do <fieldset>.
  • <label>: Etiqueta, rótulo descritivo para um campo do formulário.
    • for: Associa o rótulo do campo ao <input> correspondente.
  • <input>: Campo de entrada de dados (texto, senha, email, checkbox, radio, etc.).
    • type: Tipo de entrada (text, password, email, checkbox, radio, etc.).
    • id: Identificador único do campo.
    • name: Nome do campo enviado ao servidor.
    • value: Valor padrão do campo.
    • placeholder: Texto de exemplo dentro do campo.
    • required: indica que é um campo obrigatório.
  • <textarea>: Área de texto para várias linhas.
    • id: Identificador único do campo.
    • name: Nome do campo enviado ao servidor.
    • rows: Número de linhas visíveis.
    • cols: Número de colunas visíveis.
    • placeholder: Texto de exemplo dentro da área de texto.
  • <select>: Lista suspensa de opções.
    • id: Identificador único do campo.
    • name: Nome do campo enviado ao servidor.
  • <option>: Opção dentro de um <select>.
    • value: Valor enviado ao selecionar a opção.
    • selected: Indica a opção padrão selecionada.
  • <button>: Botão para enviar ou redefinir o formulário.
    • type: Define a função do botão (submit para enviar, reset para limpar).
    • name: Nome do botão enviado ao servidor.
    • value: Valor do botão enviado ao servidor.
    • disabled: Desabilita o botão.
    • onclick: Ação a ser executada ao clicar no botão.
